# เปลี่ยนไอคอน Shortcut

## สร้าง Shortcut บน Desktop (มีไอคอน)

ดับเบิลคลิก **`scripts\create-desktop-shortcut.vbs`** ครั้งเดียว  
จะได้ shortcut ชื่อ **SEO System** บน Desktop พร้อมไอคอน globe (โลก)

---

## เปลี่ยนไอคอนเอง

1. คลิกขวาที่ shortcut **SEO System** บน Desktop
2. เลือก **Properties**
3. กด **Change Icon...**
4. เลือกไอคอนจากรายการ หรือกด **Browse** เพื่อเลือกไฟล์ `.ico` เอง

ไอคอน Windows ที่ใช้บ่อย (ไฟล์ `C:\Windows\System32\shell32.dll`):

| ตัวเลข | ลักษณะ |
|--------|--------|
| 13 | คอมพิวเตอร์ |
| 14 | โลก (globe) |
| 21 | ดาว |
| 43 | โฟลเดอร์ |
| 269 | แอพ |

หรือใช้ไฟล์ `.ico` ที่ดาวน์โหลดมาได้
